## SCREENSHOT PROCESSING PIPELINE

Every screenshot must go through this pipeline before use:

### Step 1: Capture
- Use device screenshot (not third-party screen recorder)
- Resolution: Native device resolution (minimum 1080p)
- No status bar visible (or crop it out)
- No notification bar visible
- No personal information visible

### Step 2: Crop
- Remove status bar (top)
- Remove navigation bar (bottom)
- Keep game HUD if required for that shot
- Center the focal point using rule-of-thirds
- Ensure no awkward edge cuts

### Step 3: Resize
- Hero images: 1200px wide
- Card images: 800px wide
- Inline images: 800px wide
- Maintain aspect ratio (never stretch)

### Step 4: Optimize
- Format: WebP
- Quality: 80-85 for gameplay, 90 for text-heavy (VirusTotal)
- Use squoosh.app for manual control
- Target file sizes in the table above

### Step 5: Name
- Use exact filenames from this plan
- All lowercase, hyphens, no spaces
- Save to `public/images/real/`
