Earthkwak Games actively develops Super Bear Adventure, adding new areas, missions, collectibles, and features. Staying on an old version means missing out on new content. Check our [version changelogs](/blog/version-changelogs/) to see what has been added in recent updates.

### Mod Feature Improvements

Mod APK updates often include improvements to the mod features themselves, such as better coin rewards, new unlocked items, refined UI modifications, and compatibility patches.

### Security

Older APK files may contain vulnerabilities that have been patched in newer releases. Using the latest version reduces your exposure to potential security issues.

### Multiplayer Compatibility

If you play with friends using the [multiplayer features](/blog/multiplayer-guide/), everyone typically needs to be on the same version for online play to work correctly. An outdated version can prevent you from connecting to other players.

## Step 1: Back Up Your Save Data

This is the most critical step. Skipping the backup is the primary reason players lose progress during updates. We recommend using at least one of the methods below, and ideally both for extra safety.

### Understanding Where Save Data Is Stored

Super Bear Adventure stores save data locally on your device. The typical location is:

```
Android/data/com.Earthkwak.Platformer3D/files/
```

Or in some versions:

```
/storage/emulated/0/Android/data/com.Earthkwak.Platformer3D/
```

The exact path may vary slightly depending on your device and Android version. The save data includes your mission progress, collected stickers, unlocked skins and hats, coin balance, and world completion status.

### Method A: Manual File Backup (Recommended)

This gives you direct control over your save files.

1. **Install a file manager**: If you do not already have one, install a file manager app like "Files by Google," "Solid Explorer," or "ZArchiver" from the Play Store. Make sure it can access the Android/data folder (on Android 11 and later, some file managers have restricted access to this folder).

2. **Navigate to the save data folder**: Open your file manager and go to:
   ```
   Internal Storage > Android > data > com.Earthkwak.Platformer3D > files
   ```

3. **Copy the entire folder**: Select the `files` folder (or its contents) and copy it to a safe location. Good options include:
   - A folder on your internal storage outside the Android/data directory (such as a "Backups" folder)
   - Your device's Downloads folder
   - An external SD card
   - A cloud storage service (Google Drive, Dropbox, etc.)

4. **Verify the backup**: Open the backup location and confirm the files are there. Do not skip this step.

#### Handling Android 11+ File Access Restrictions

Starting with Android 11, Google restricted access to the Android/data folder. If your file manager cannot open this folder, try these solutions:

- **Use Solid Explorer or MiXplorer**: These file managers support SAF (Storage Access Framework) and can be granted access to restricted directories. When prompted, grant access to the Android/data folder.
- **Connect to a computer**: Plug your phone into a PC via USB, enable file transfer mode, and navigate to the folder using your computer's file explorer. On some devices, you may still face restrictions; in that case, use ADB (covered below).
- **Use ADB**: Connect your device to a PC, open a command prompt, and use:
  ```
  adb pull /storage/emulated/0/Android/data/com.Earthkwak.Platformer3D/ ./backup/
  ```
  This copies the entire game data folder to your computer.

### Method B: App-Based Backup

Several backup apps can automate this process.

1. **Swift Backup** or **Migrate**: These apps can back up both the APK and its associated data. Install one from the Play Store.

2. **Create a backup**: Open the backup app, find Super Bear Adventure in the app list, and create a full backup (app + data).

3. **Store the backup safely**: Move the backup file to cloud storage or an external location for safekeeping.

4. **Note**: Some backup apps require root access for full data backup. If your device is not rooted, the manual file method may be more reliable.

### Method C: Cloud Sync (If Available)

Some newer versions of Super Bear Adventure may include built-in cloud save functionality. If this feature is available in your version:

1. Open the game and go to Settings.
2. Look for a Cloud Save or Sync option.
3. Sign in with your Google account if prompted.
4. Manually trigger a sync to upload your current progress.

This method is the easiest but is not available in all versions, particularly older mod versions.

## Step 3: Install the Update (Without Uninstalling First)

This is where most players make mistakes. The key rule is: **do not uninstall the old version before installing the new one**, unless you absolutely have to.

### The Correct Update Process

1. **Open the downloaded APK file**: Use your file manager or tap the download notification to open the new APK file.

2. **Allow installation from unknown sources** (if prompted): On Android 8 and later, you need to grant this permission to the specific app you are using to install the APK (your file manager or browser). Go to Settings > Apps > [Your file manager/browser] > Install Unknown Apps > Allow.

3. **Tap Install**: Android will detect that a previous version of the app already exists. It should show a prompt indicating that the existing app will be updated. This is what you want.

4. **Wait for installation**: The process usually takes 10-30 seconds.

5. **Open the game**: Launch Super Bear Adventure and verify that your progress is intact. Check your coin balance, unlocked [skins and hats](/blog/unlock-all-skins-hats/), and mission completion status.

### Why This Preserves Your Data

When you install a new APK over an existing one (an "overlay" or "upgrade" install), Android keeps the app's data directory intact. The old app code is replaced with the new version, but the saved games, preferences, and other data files remain untouched. This only works when:

- The new APK has the same package name as the old one (both should be `com.Earthkwak.Platformer3D` or the mod's package name).
- The new APK is signed with the same certificate as the old one.

If either condition is not met, Android will refuse the overlay install and require you to uninstall first. This is when your backup becomes essential.

## Step 4: What to Do If the Update Fails

Sometimes the overlay install does not work. Here are the most common failure scenarios and how to handle each one.

### Error: "App Not Installed" or "Installation Failed"

This usually means there is a signature mismatch between the old and new APK files. The new mod version may have been signed with a different certificate than the one you currently have installed.

**Solution**:

1. Do not panic. Your backup from Step 1 has your save data safe.
2. Uninstall the current version of the game.
3. Install the new APK file.
4. Restore your save data from the backup (see Step 5 below).

### Error: "There Was a Problem Parsing the Package"

This means the APK file is corrupted or incompatible with your Android version.

**Solution**:

1. Delete the downloaded APK file.
2. Re-download it from the source.
3. Check that your Android version meets the minimum requirement listed on the download page.
4. Try a different browser for downloading (Chrome sometimes handles APK downloads better than others).

### Error: "Insufficient Storage"

Your device does not have enough free space to complete the installation.

**Solution**:

1. Free up storage by deleting unused apps, clearing cache, or moving media files to cloud storage.
2. You need roughly 300-500 MB of free space for the installation process (the game file plus temporary installation space).
3. Retry the installation.

### Error: "Install Blocked" or "Security Policy Prevents Installation"

Your device settings are preventing the installation.

**Solution**:

1. Go to Settings > Apps > [your file manager] > Install Unknown Apps and enable the permission.
2. If you have a work profile or enterprise management on your device, those policies may block sideloaded apps. You may need to use a personal device instead.
3. Check that no security apps (antivirus, device management) are blocking the installation.

For more error solutions, visit our [troubleshooting guide](/blog/troubleshooting-common-errors/).

## Step 5: Restoring Save Data After a Clean Install

If you had to uninstall the old version before installing the new one, follow these steps to restore your backed-up save data.

### Restoring from Manual Backup

1. Install and open the new mod APK. Launch the game once and reach the main menu, then close it. This creates the necessary data folders.

2. Open your file manager and navigate to the backup location where you saved your files in Step 1.

3. Copy the backed-up save files.

4. Navigate to the game's data folder:
   ```
   Android/data/com.Earthkwak.Platformer3D/files/
   ```

5. Paste the backup files, overwriting any new files that were just created.

6. Launch the game. Your progress should be restored.

### Restoring from App-Based Backup

1. Open your backup app (Swift Backup, Migrate, etc.).
2. Find the Super Bear Adventure backup in your backup list.
3. Choose to restore **data only** (not the app, since you already installed the new version).
4. Launch the game and verify your progress.

### Restoring via ADB

If you backed up using ADB on a computer:

1. Connect your device to the PC.
2. Open a command prompt and run:
   ```
   adb push ./backup/ /storage/emulated/0/Android/data/com.Earthkwak.Platformer3D/
   ```
3. Launch the game and check your progress.

## When to Do a Clean Install Instead

In some cases, a clean install (uninstalling the old version and starting fresh) is actually the better choice. Consider a clean install when:

- **You are switching from the official Play Store version to the mod** (or vice versa). The package names or signatures are likely different, so an overlay install will not work.
- **The game is severely bugged on your current installation**. Corrupted data can carry over through updates. A fresh start with restored save files can fix persistent issues.
- **You are jumping several major versions ahead**. If your current version is very old and the new version has undergone significant changes, a clean install ensures compatibility.
- **Save data restoration does not work**. If you have tried restoring your backup and the game does not recognize it, the save format may have changed between versions. In this case, starting fresh may be your only option.

Even with a clean install, always back up first. You can attempt to restore after installing, and if it does not work, at least you still have the backup files for future reference. ## Frequently Asked Questions

### Will I lose my coins if I update?

Not if you follow this guide. Installing the new APK over the old one (without uninstalling first) preserves all data, including your coin balance. If you must uninstall first, your backup will protect your coins.

### Can I update directly from the Play Store?

If you are using the mod APK, no. The Play Store version and the mod version have different signatures. The Play Store will not recognize the mod as an updatable version of the game. You need to download the mod APK update manually.

### What if the new version crashes on startup?

First, try clearing the game's cache (Settings > Apps > Super Bear Adventure > Clear Cache). If that does not help, the issue may be incompatible save data from an older version. Try a clean install and restore your backup. If crashes persist, check our [troubleshooting guide](/blog/troubleshooting-common-errors/) or wait for a patch.

### How do I know if my save data was backed up correctly?

After copying your save files, navigate to the backup location and check that files exist there. The backup folder should contain files and should not be empty. If you can, compare the file sizes to the originals to make sure the copy completed fully.

### Can I update from a very old version to the newest one?

Generally yes, but the larger the version gap, the higher the chance of compatibility issues. If you are many versions behind, consider doing a clean install and restoring save data. If save data restoration fails due to format changes, you may need to start fresh.

### Do I need to re-configure anything after updating?

In most cases, your settings, control preferences, and display options will carry over with the update. Occasionally, a major update may reset some settings to defaults. After updating, quickly check your settings menu to make sure everything is as you prefer.

### Is it safe to update over Wi-Fi or should I use mobile data?

The download source matters more than the connection type. Use whatever connection is most stable to avoid incomplete downloads. If you are on mobile data, make sure you have sufficient data allowance, as the APK file can be 100-200 MB or more.

### Can I use the same backup for different mod versions?

Save data backups are generally compatible across versions of the same game, but not always. If you switch between significantly different mod builds (especially from different sources with different package names), your backup may not be recognized. Always back up again before switching to a different mod source.
