## How to Downgrade Safely Without Losing Progress

Downgrading carries risk to save data. Cloud saves from newer versions may not load in older ones, and major version jumps almost certainly change the save file format. Minor downgrades (v10.5 to v10.2) are more likely to preserve compatibility.

**Recommended downgrade path:**

1. Back up your save data to a separate folder on your device or PC
2. Uninstall the current version
3. Install the target older version
4. Launch the game and check if it creates a new save or detects your backup
5. If your progress is lost, you will need to start fresh on that version

To keep your current progress while trying an old version, consider using a secondary device or an Android emulator.

## Frequently Asked Questions

### Can I transfer my save data between different versions?

Save data can sometimes transfer between minor version changes (like v10.2 to v10.5), but major version jumps usually break save compatibility. Always back up before attempting a version change.

### Will multiplayer work if I am on a different version than my friend?

No. Multiplayer requires both players to be on the same version. If your friend is on v11 and you are on v10, you will not be able to connect. Our [multiplayer guide](/blog/multiplayer-guide/) has more details on connection requirements.

### Is it safe to install old APK versions?

Older versions from trusted sources are generally safe to install. However, they will not have the latest security patches. Avoid downloading APKs from unverified websites, as they may be modified with malware.

### Why does the game crash after I install an old version?

This usually happens when leftover data from a newer version conflicts with the older build. Try clearing the app's cache and data after installing, or do a completely clean install by removing all associated files first. Our [troubleshooting guide](/blog/troubleshooting-common-errors/) covers crash solutions in detail.

### Can I have two different versions installed at the same time?

Not without modification. Android identifies apps by their package name, so installing a second version will overwrite the first. Some users work around this with app cloning tools or by using an emulator for the second version.

### Which version has the best balance between content and performance?

For most players, v10.2.0 hits the sweet spot. It includes Turtletown, Snow Valley, Beemothep Desert, and Giant House, while being lighter than the v11.x builds that add Mole Mines overhead.

### How do I know which version a specific mod requires?

Check the mod's documentation or download page. Most mod creators specify the target game version. If not stated, try the latest version first and work backward if you encounter issues.
