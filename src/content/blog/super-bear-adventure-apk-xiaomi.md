Xiaomi devices running MIUI or HyperOS have a unique security flow for sideloaded APKs compared to stock Android. The MIUI Security app and a stricter unknown sources path add a few extra steps. This guide covers all Xiaomi sub-brands: Xiaomi, Redmi, POCO.

## Compatible Xiaomi Devices

Super Bear Adventure Mod APK v12.1.5 works on all Xiaomi devices running Android 5.1 or later:

**Xiaomi (Flagship):**
- Xiaomi 14, Xiaomi 14 Ultra
- Xiaomi 13, Xiaomi 13 Pro
- Xiaomi 12 series and older

**Redmi (Mid-range):**
- Redmi Note 13, Note 13 Pro, Note 13 Pro+
- Redmi Note 12 series
- Redmi 13C, Redmi 12, Redmi A3

**POCO:**
- POCO X6, X6 Pro
- POCO F6, F5
- POCO M6, M4

**Operating Systems:** MIUI 12, 13, 14, and HyperOS — all compatible.

## MIUI vs HyperOS: What Differs for APK Install

**MIUI 12–14:** The Security app is deeply integrated and aggressively flags sideloaded APKs. The unknown sources path is buried under Privacy → Special Permissions.

**HyperOS:** Xiaomi's replacement for MIUI, rolling out to flagship models from late 2023. The security flow is slightly less aggressive but still requires the same steps. The Settings path for unknown sources is identical.

## Step-by-Step Install on Xiaomi/MIUI

### Step 1: Download the APK

Open **Chrome** or **Mi Browser** and navigate to [superbearadventuremod.io/download/](/download/). Tap Download.

**Recommendation:** Use Chrome rather than Mi Browser if MIUI version is 12 or 13 — Mi Browser in these versions has its own security filter that may block the download before it starts.

### Step 2: Enable Unknown Apps

The path varies by MIUI version:

**MIUI 14 / HyperOS:**
Settings → Privacy → Special Permissions → Install Unknown Apps → [your browser] → Allow

**MIUI 12 / 13:**
Settings → Special App Permissions → Install Unknown Apps → [your browser] → Allow

**Older MIUI versions:**
Settings → Additional Settings → Privacy → Install Apps from Unknown Sources → toggle ON

### Step 3: Handle the MIUI Security Scan

When you tap the APK, MIUI Security automatically scans it. You will see one of two results:

**Scenario A — "No risks found":**
Proceed normally to the install prompt.

**Scenario B — "Risk detected" or "Security scan":**
This is MIUI's standard warning for all unsigned APKs. It does not mean the file contains actual threats. Tap **Install Anyway** (or **Proceed** on some MIUI versions).

To disable the automatic scan for this install only: Open **MIUI Security** app → go to **Settings** → disable **Auto Scan** temporarily. Re-enable after install.

### Step 4: Install

1. Tap **Install** in the Android install dialog
2. If MIUI shows "Blocked by MIUI", go to Security → System Apps Manager → locate the APK installer → Enable
3. Installation takes 10–30 seconds
4. Tap **Open** to launch immediately

## Common MIUI-Specific Issues

### "Miui has blocked the installation of this application"

This blocking message appears on MIUI 12 in some regions where Xiaomi's server-based verification is enabled. Solutions:

1. **Disable Mi Security temporarily:** Open Security app → Settings → disable cloud verification
2. **Install via ADB sideload** (advanced users): Connect to PC, enable USB Debugging, run `adb install super-bear-adventure-mod-apk.apk`
3. **Switch network:** Some users report the block is network-dependent; try on a different Wi-Fi network

### "Package not installed" on HyperOS

If the APK installs but the game does not appear in the app drawer, clear the launcher cache:
Settings → Apps → Launcher → Storage → Clear Cache → restart launcher.

### Game Crashes After Launch on Xiaomi

1. Settings → Apps → Super Bear Adventure → Permissions → ensure all requested permissions are granted
2. Settings → Battery → Battery Optimization → find Super Bear Adventure → set to "No Restrictions"
3. Settings → Apps → Super Bear Adventure → Autostart → Enable

The Autostart setting is a MIUI-specific restriction. Enabling it prevents MIUI from killing the game in the background during multiplayer sessions.

### "Permission Denied" Installing from Mi Browser

Mi Browser on MIUI 13+ has its own APK download security layer. If it blocks the download:
1. Copy the download URL from our [download page](/download/)
2. Open Chrome
3. Paste the URL and download from Chrome instead

## Autostart and Battery Optimization (Important for MIUI)

After installing, configure these MIUI-specific settings for best performance:

**Autostart:**
Settings → Apps → Manage Apps → Super Bear Adventure → toggle Autostart ON

**Battery Optimization:**
Settings → Battery & Performance → App Battery Saver → Super Bear Adventure → set to No Restrictions

**Background Activity:**
Settings → Apps → Super Bear Adventure → Data Usage → enable Background Data

These settings prevent MIUI from aggressively killing the game during multiplayer sessions.
