## What Changed in Android 14 for APK Installs

Android 13 introduced Restricted Settings and Android 14 tightened the unknown sources flow. Here is what actually changed:

### Unknown Sources Is Now Per-App

In Android 9 and earlier, "Unknown Sources" was a single toggle in Security settings. Since Android 10, it is per-app — each app that downloads APKs needs its own permission. On Android 14, this setting is at:

**Settings → Apps → Special App Access → Install Unknown Apps**

You need to find the specific app you used to download (usually Chrome, Firefox, or a file manager) and enable it there.

### Restricted Settings

Android 13 introduced Restricted Settings, which prevents apps installed from unknown sources from accessing accessibility services or notification listener permissions. This is relevant for certain mod menus that use accessibility shortcuts.

For Super Bear Adventure Mod APK specifically, Restricted Settings **does not block the install or gameplay** — the game does not require accessibility permissions.

### Android 15 Play Protect Prompt

Android 15 may show an additional Play Protect warning for APKs from unknown sources. This is a standard warning — tap "Install Anyway" to proceed. The mod APK is clean and the warning is not indicative of any actual threat.

## Android 14-Specific Device Issues

### Pixel 8 / Pixel 9

Google Pixel devices on Android 14+ enforce Play Protect most strictly. If Play Protect blocks the install:

1. Open **Google Play Store**
2. Tap your profile icon → **Play Protect**
3. Tap the gear icon → disable **Scan apps with Play Protect** (temporarily)
4. Install the APK
5. Re-enable Play Protect after install

The game will run correctly with Play Protect re-enabled — the runtime scan after install will pass.

### OnePlus / OxygenOS 14

OxygenOS 14 combines Restricted Settings with additional OEM security. Go to:

**Settings → Security → More Security Settings → Install Unknown Apps**

The path differs slightly from stock Android. Find your browser in this menu.

### Motorola / Android 14

Motorola devices running Android 14 have a streamlined unknown sources flow. When you tap the APK, the system automatically asks if you want to enable the permission for that source — tap Allow and installation proceeds immediately.

## Android 15 Differences

1. Enable unknown sources per-app (same location as Android 14)
2. Download and tap the APK
