## Step-by-Step Setup

### Step 1: Download and Install LDPlayer 9

Download LDPlayer from ldplayer.net (Windows only). Run the installer. LDPlayer installs a virtual Android 9 environment in about 5 minutes.

### Step 2: Download the Mod APK to Your PC

Download the APK from [superbearadventuremod.io/download/](/download/) on your PC browser. Save it to your Desktop or Downloads folder.

File name: `super-bear-adventure-mod-apk.apk` (221 MB)

### Step 3: Install the APK in LDPlayer

**Method A (Drag and Drop):**
Drag the APK file from your Desktop onto the LDPlayer window. LDPlayer opens the APK Installer automatically.

**Method B (APK Installer Button):**
In LDPlayer, look at the right sidebar — click the **APK file icon** (looks like a folder with an arrow). Browse to your APK file and click Open. The installer runs automatically.

**Method C (Right-click):**
Right-click the APK on your Desktop → Open with → LDPlayer (if associated).

Click **Install** in the dialog. Installation takes 15–30 seconds.

### Step 4: Launch the Game

After install, find Super Bear Adventure in the LDPlayer app drawer (click the square grid icon to see all installed apps). Click to launch.

First launch: game loads all shaders and assets, which takes 30–60 seconds. Subsequent launches are much faster.

### Step 5: Configure Keyboard Controls

1. Click the **keyboard icon** in LDPlayer's right sidebar
2. Select **Keyboard Mapping**
3. LDPlayer shows an overlay of your game's screen
4. **Drag control types** onto on-screen buttons:
   - Drag a "WASD" control onto the movement stick area
   - Drag a "Tap" button onto Jump, Attack, Interact
   - Drag a "Zoom" control onto the camera if needed
5. Click **Save**

**Recommended key layout:**
- WASD or Arrow Keys: Movement
- Space: Jump
- Left Click: Attack
- E or F: Interact/Action
- Q: Dodge (if applicable)
- Escape: Menu

LDPlayer also supports USB gamepads if you prefer controller play.

## Performance Settings in LDPlayer

### CPU and RAM Allocation

Right-click the LDPlayer icon in the taskbar → Settings → Performance:
- CPU: 4 cores (or all available)
- Memory: 4096 MB (4GB) minimum

### FPS Target

Settings → Display → Frame Rate → 60 (or 120 for high-end setups)

### Graphics Mode

Settings → Other Settings → Rendering Mode:
- **DirectX (recommended)** — best compatibility
- **OpenGL** — better for older GPUs

### Resolution

Settings → Display → Resolution: 1280×720 is the sweet spot. Higher resolutions look better but demand more GPU.

## Multi-Instance Play

LDPlayer supports multiple emulator instances simultaneously. This is relevant if you want to test multiplayer co-op:

1. In LDPlayer → Instance Manager → New Instance
2. Create a second LDPlayer instance
3. Install the mod APK on both instances
4. Start a multiplayer session on one → use invite code to join on the second

This simulates the 4-player co-op experience on a single PC.

## Common LDPlayer Issues

| Issue | Solution |
|-------|----------|
| LDPlayer won't start | Enable VT-x or AMD-V in BIOS |
| APK fails to install | Ensure LDPlayer is fully loaded before installing — wait for the home screen |
| Low FPS | Settings → Performance → increase RAM and CPU cores |
| Game crashes at launch | Clear LDPlayer app cache: LDPlayer → Apps → Super Bear Adventure → Clear Cache |
| Black screen on launch | Switch rendering mode: Settings → Other → Rendering Mode → DirectX ↔ OpenGL |
| Keyboard controls not working | Ensure keyboard mapping was saved and the game window is focused |
